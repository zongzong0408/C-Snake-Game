## 開發環境

- Windows Home 10 64bit
- `gcc 6.3.0` (MinGW.org GCC-6.3.0-1)
- DevC++
- CodeBlocks 17.12
- `C99`

---

## 如何執行

> 預設 ***Windows Home 10 64bit*** 作業系統  

- 方法一：直接點兩下執行 `snake.exe`

- 方法二：在本地直接使用 `gcc` 編譯 `snake.c` 

```bash
gcc -o snake.exe snake.c
```
```bash
snake.exe
```
